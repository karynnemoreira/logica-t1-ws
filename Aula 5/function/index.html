Function

Uma função é um bloco de código que realiza uma tarefa especifíca e pode ser reutilizado.

Funções podem receber dados externos através de parâmetros() . Esses dados são como variáveis que a função utiliza ao ser chamada.

  • function: palavra-chave usada para criar a função.
  • nomeDaFuncao: Esse é o nome da função. Geralmente, escolhemos um nome que descreva o que a função faz, como mostrarMensagem ou calcularSoma.
  • Parênteses (): É onde colocamos os parâmetros (ou seja, os valores que a função recebe).
  • Chaves {}: Dentro das chaves fica o código que a função vai executar.

Parâmetros

Os parâmetros de uma função são os valores que a função espera receber.

Argumentos

Os argumentos são os valores que são passados para uma função, preenchendo os marcadores de posição definidos pelos parâmetros.

Função Anônima

É uma função que não tem um nome. Ou seja, você cria a função, mas não dá um nome para ela.

Quando usamos função anônima?

Quando queremos criar algo rápido e direto, sem precisar dar um nome para a função. Muitas vezes, a gente usa elas dentro de outras funções.

Arrow Function

Uma expressão de função de seta é uma alternativa compacta a uma expressão de função tradicional
