Variáveis

Let: Declara uma variável local (Escopo local, é declarada dentro de uma função) Pode ser redeclarada , podem mudar seu valor.

Const: Não pode ser atualizada, tem o mesmo valor (é constante) Use const para valores que não mudam.

Var: Podem ser atualizadas e declaradas dentro do seu escopo. (Escopo global, quando é declarada fora de uma função) É uma variável antiga, menos utilizada atualmente.
